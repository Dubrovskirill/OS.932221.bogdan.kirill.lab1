
# Вывод текущего расстояния до космического аппарата «Вояджер-2» 

## Команды для работы в терминале

Все команды должны быть выполнены в директории, где находятся файлы `main.c` и `Makefile`.

### Сборка модуля (Компиляция)

Перед сборкой убедитесь, что в `Makefile` указан `main.o` (так как ваш исходник называется `main.c`) и что установлены заголовочные файлы ядра (`linux-headers-$(uname -r)`).

**Последовательность выполнения:**

```bash
make clean
make
sudo insmod main.ko
sudo dmesg
cat /proc/tsulab
sudo rmmod main
sudo dmesg

````
