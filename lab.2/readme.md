# Лабораторная работа 2


## Сборка

1.  Откройте терминал и перейдите в директорию с файлом `main.c`.
2.  Выполните команду для компиляции:
    ```bash
    gcc -Wall -Wextra -o server main.c
    ```
    Это создаст исполняемый файл `server`.

## Запуск

 После успешной сборки запустите сервер:
    ```bash
    ./server
    ```
    Сервер начнёт прослушивание порта `31337`.

## Тестирование

1.  **Подключение клиента:** В *новом* терминале подключитесь к серверу, например, с помощью `telnet`:
    ```bash
    telnet localhost 31337
    ```
    В терминале с запущенным `./server` должно появиться сообщение о новом соединении. Сервер принимает *одно* соединение. Любые попытки подключиться снова будут приводить к мгновенному закрытию нового соединения сервером.

2.  **Отправка данных:** В окне `telnet` введите текст и нажмите `Enter`. В терминале с `./server` должно появиться сообщение о получении данных с указанием количества байт.

3.  **Отправка сигнала `SIGHUP`:** В *еще одном* терминале найдите PID процесса `server` (например, `ps aux | grep server`) и отправьте ему сигнал `SIGHUP`:
    ```bash
    kill -SIGHUP <PID_сервера>
    ```
    Альтернативно, можно использовать `killall -SIGHUP server`, если запущен только один экземпляр.
    Сервер выведет сообщение о получении сигнала и завершит работу.
